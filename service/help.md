##一、环境安装：
环境要求：
  node -v
v8.9.3
  npm -v
5.5.1

##二、项目搭建
第一步: 安装koa-generator
npm install koa-generator -g
第二步：使用koa-generator生成koa2项目
// 命令：koa2 "项目名称"，我们可以把项目名字为：blog
koa2 blog
预览
第三步：安装包
// 进入blog项目
cd blog
// 安装服务
npm install
第四步：启动服务
npm start
